# House Sales Analysis – King County, USA

## Overview

This project analyzes residential house sales data from King County, USA. The objective is to explore pricing patterns, understand the relationship between housing features and sale prices, and apply basic data analysis and visualization techniques using Python.

This repository is intended to demonstrate foundational data analysis skills, including data cleaning, exploratory data analysis (EDA), and simple modeling. It is suitable for early‑career data analysts and as a portfolio project for interviews.

## Dataset

* **Source**: House Sales in King County, USA
* **Format**: CSV (loaded and analyzed in a Jupyter Notebook)
* **Content**: The dataset includes variables such as house price, number of bedrooms and bathrooms, square footage, location attributes, and sale dates.

## Tools & Technologies

* Python
* Jupyter Notebook (JupyterLite compatible)
* pandas
* numpy
* matplotlib
* seaborn
* scikit‑learn (Linear Regression)

## Key Analysis Steps

1. **Data Loading** – Importing the dataset and inspecting structure and data types.
2. **Data Cleaning** – Handling missing values, verifying data consistency, and preparing features for analysis.
3. **Exploratory Data Analysis (EDA)** –

   * Summary statistics
   * Distribution analysis
   * Correlation analysis
   * Data visualizations
4. **Modeling** – Applying linear regression to understand relationships between features and house prices.
5. **Insights & Interpretation** – Interpreting trends and results in a business‑relevant context.
